---
description: >-
  TFHE-rs is a pure Rust implementation of TFHE for Boolean and integer
  arithmetics over encrypted data. It includes a Rust and C API, as well as a
  client-side WASM API.
---

### References & Explanations

Take a deep dive into TFHE-rs, exploring APIs from the highest to the lowest level of abstraction and accessing additional resources for in-depth explanations&#x20;

* [API references](https://docs.rs/tfhe/latest/tfhe/): High-level API that abstracts cryptographic complexities and simplifies the development
* [Fine-grained APIs](references/fine-grained-apis/): Mid-level API that enables evaluation of Boolean, short integer, and integer circuits
* [Crypto core API](references/crypto-core-api/): Low-level API with the primitive functions and types of the TFHE scheme
* [TFHE deep dive](explanations/tfhe-deep-dive.md): Resources that explain the Fully Homomorphic Encryption scheme - TFHE
